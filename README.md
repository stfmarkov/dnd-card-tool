# item-cards

**item-cards** is a small utility for **designing and printing item cards** for *Dungeons and Dragons* at the table. You add a name, type line, description, and artwork, see a book-style card preview, then print the result when you are ready. It is built for home games and one-off handouts.

A downloadable release will be published later. This repository is mainly for the project and people who work on it.

This project is a fan work and is not affiliated with or endorsed by Wizards of the Coast.
